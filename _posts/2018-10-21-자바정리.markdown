# 새로운 저장소 만들기

폴더를 하나 만들고, 그 안에서 아래 명령을 실행 ###$ git init 새로운 git 저장소가 만들어진다.

----------

# 저장소 받아오기

로컬 저장소를 복제(clone)하려면 아래 명령을 실행 ###$ git clone /로컬/저장소/경로 원격 서버의 저장소를 복제하려면 아래 명령을 실행하세요. ###$ git clone 사용자명@호스트:/원격/저장소/경로
