## Linux 명령어  
mkdir : 디렉토리 생성 ( 폴더 생성 ) == 앞으로 폴더를 디렉토리로 쓰겠습니다.  
새로운 파일 생성하기 - touch    
새로운 폴더 생성하기 - mkdir == 현재 폴더 내에 새로운 폴더를 생성합니다.  

## cd 명령어 - 디렉토리 이동  
cd == 이는 해당 디렉토리로 이동하는 명령어이다. 만약 해당 디렉토리가 있다면 해당 디렉토리로 이동을 하지만 없다면 다음과 같이 에러 메시지를 출력한다.  
cd . == 현재 디렉토리.  
cd .. == 해당 디렉토리에서 상위 디렉토리로 이동.  
cd / == 최상위 디렉토리로 이동.  
cd \$ 변수명 == 변수에 저장된 경로로 이동.  
cd ~ ,cd \$HOME, cd == 사용자 홈 디렉토리로 이동  
cd ~계정명 == 입력한 사용자의 홈 디렉토리로 이동  
cd - == 이전 경로로 이동
